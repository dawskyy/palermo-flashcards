PALERMO FLASHCARDS (Italian Learning App in Console)

Very simple, console flashcards app written in Python. I created this project to help me learn Italian vocabulary before my upcoming trip to Palermo. It's a great practical coding exercise that also serves a real-life purpose.

Key features and what I've practiced:

- Speech Synthesis (gTTS): Integrated audio so you can actually hear the Italian pronunciation.
- 3 Learning Modes: Added PL-IT, IT-PL, and a new "Gap Filling" mode to practice words in full sentences.
- OS Detection: Added a basic check for Windows/Linux to handle audio playback automatically on both systems.
- Database Update: Improved vocabulary management and progress tracking (JSON).
