PALERMO FLASHCARDS (Italian Learning App in Console)

Very simple, console flashcards app written in Python. I created this project to help me learn Italian vocabulary before my upcoming trip to Palermo. It's a great practical coding exercise that also serves a real-life purpose.

Key features and what I've practiced:
- Storing and managing vocabulary data.
- User input handling and verifying correct answers.
- Randomizing the order of displayed words.
- Keeping track of the score/progress during the session (sometimes it's working fr).

PALERMO FLASHCARDS (Konsolowa aplikacja do nauki włoskiego)

Bardzo prosta, konsolowa aplikacja z fiszkami napisana w Pythonie. Stworzyłem ten projekt, żeby pomóc sobie w nauce włoskich słówek przed nadchodzącym wylotem do Palermo. To świetne, praktyczne ćwiczenie z programowania, które przy okazji ma realne zastosowanie.

Główne funkcjonalności i co przećwiczyłem:
- Przechowywanie i zarządzanie bazą słówek.
- Obsługa danych wejściowych od użytkownika i weryfikowanie poprawnych odpowiedzi.
- Losowanie kolejności wyświetlanych słów.
- Śledzenie punktacji/postępów w trakcie sesji (czasami nawet działa, serio xd).
